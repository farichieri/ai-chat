# Create Project

```bash
npx create-next-app
```

## Install Dependencies

```bash
pnpm add @ai-sdk/openai # sdk for integrating OpenAI's API

pnpm add nanoid # generate unique string ID generator for messages

# Styling
pnpm add rehype-raw remark-gfm react-markdown # Markdown content in React

pnpm dlx shadcn@latest init
pnpm dlx shadcn@latest add button
pnpm dlx shadcn@latest add textarea

pnpm add @tailwindcss/typography # Pluging for simple typography
```
